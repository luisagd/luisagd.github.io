# Magnitudes

Una magnitud es todo aquello que se puede medir en una escala y con un instrumento adecuado. Es decir, todo aquello que se puede medir.

## Clasificacion de las Magnitudes

- Según su maturaleza:

  - Escalares: Aquelllos que pueden definirse con el valor numérico y la unidad de medida correspondiente. Ej: masa, tiempo, longitud\*, temperatura.

  - Vectoriales: Aquellas que, además de necesitar de valor numérico y la unidad de medida, necesitan de dirección y sentido para definirse.
    Ej: Fuerza, velocidad, aceleración, longitud \*.

- Según el Sistema de unidades:

  - Fundamentales: Son las magnitudes básicas a partir de las cuales derivan todas las demás magnitudes del sistema.

  - Derivadas: Son las que derivan de las fundamentales.

# Vector

Es un segmento de recta orientado, herramienta matemática utilizada para el estudio de las magnitudes vectoriales.
